  • Apa Saja Yang Harus Dibuat Untuk Tugas Besar?

    Untuk Tugas Besar yang harus disiapkan adalah presentasi per tiap minggunya, serta laporan tugas besar, dan semua file tugas besar dapat diupload di form pengumpulan. jangan lupa progres terus diupdate di Logbook website ini ya.

  • Ukuran Laporan?

    Halo, Spesifikasi ukuran laporan dapat ditanyakan langsung kedosen yang bersangkutan. Karena tiap dosen pengampu berbeda spesifikasi ukuran laporannya:) Kontak dosen bisa dilihat dipertanyaan pertama.

  • Spesifikasi Presentasi?

    Halo, Spesifikasi presentasi dapat ditanyakan langsung kedosen yang bersangkutan. Karena tiap dosen pengampu berbeda spesifikasi presentasinya:) Kontak dosen bisa dilihat dipertanyaan pertama.

  • Apakah File Presentasi Dikumpulkan?

    Halo, Untuk file presentasi dapat dikumpulkan pada form pengumpulan tugas besar yang telah disediakan.

  • Bagaimana Saya Mendapatkan Referensi Tugas Besar?

    Halo, Pada Website Sistem Informasi Tugas Besar ini sudah menyediakan arsip arsip tugas besar serta data data yang diperlukan. Silahkan klik tombol Arsip / Data pada website ini. Terima Kasih.
